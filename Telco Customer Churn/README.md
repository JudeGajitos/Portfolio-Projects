# **Telco Customer Churn Analysis**
This project analyzes customer churn behavior in a telecommunications company to identify patterns and factors associated with customer attrition.

# **OBECTIVES**
- Identify services associated with high churn
- Analyze the impact of contract type on churn
- Provide business-driven insights to reduce churn

# **INSIGHTS**
- Fiber optic internet service records the highest number of churned customers (1,104).
- Customers on month-to-month contracts exhibit higher churn compared to one-year and two-year contracts.
- Higher monthly charges increase churn probability.

# **TOOLS USED IN THIS PROJECT**
- Python (Pandas, Numpy)
- Matplotlib and Seaborn
- Google Colab
